## Documentation Rule

새 컴포넌트를 추가할 때는 아래 항목을 기록합니다.

- 목적
- 주요 props
- 사용 예시
- 상태: default, loading, disabled, error, empty
- 접근성 고려사항
- `/storybook` 예시 여부

# UI Components

## Basic

| Component | Description |
| --- | --- |
| `Button` | 버튼, 로딩, 아이콘 지원 |
| `Input` | 좌/우 아이콘 슬롯 지원 |
| `Textarea` | 여러 줄 입력 |
| `Select` | 기본 선택 입력 |
| `Checkbox` | 체크박스와 설명 |
| `RadioGroup` | 라디오 그룹 |
| `Switch` | 토글 |
| `Badge` | 상태 표시 |
| `Card` | 콘텐츠 컨테이너 |
| `Typography` | 텍스트 스타일 |

## Overlay

| Component | Description |
| --- | --- |
| `Modal` | 중앙 모달 |
| `Drawer` | 측면 패널 |
| `Popover` | 트리거 기반 보조 패널 |
| `Dropdown` | 액션 메뉴 |
| `CommandPalette` | 명령 검색 UI |
| `Dialog` | alert/confirm/prompt 대체 |
| `Toast` | 전역 알림 |

## Data / Navigation

| Component | Description |
| --- | --- |
| `Table` | 기본 테이블 |
| `DataTable` | empty/loading/pagination 포함 테이블 |
| `Pagination` | 페이지 이동 |
| `Tabs` | 탭 |
| `Accordion` | 접이식 패널 |
| `TreeView` | 계층 데이터 |
| `Stepper` | 단계 표시 |

## Feedback

| Component | Description |
| --- | --- |
| `Alert` | 인라인 알림 |
| `Snackbar` | 하단 알림 |
| `Spinner` | 로딩 스피너 |
| `Skeleton` | 로딩 플레이스홀더 |
| `Progress` | 진행률 |
| `EmptyState` | 빈 상태 |
| `ErrorState` | 오류 상태 |
| `LoadingState` | 로딩 상태 |
